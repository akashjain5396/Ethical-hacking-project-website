BANDIT: 01

The user need to know about commands [ls] and [cat]. [ls] gives the list of all the files in the directory and [cat] is used to print a file on the standard output. You can see about them on the manual page of them by typing following commands on terminal.

$ man ls

$ man cat

Commands:

$ ls

You will see a single file naming readme. Read it.

$ cat readme

You will see the password to the next level.
