BANDIT: 13

You need to know about different types of file compression and hexdump file. You also need to know how to make directories.

Commands:

$ mkdir /tmp/myname/

This will create a directory in /tmp of name myname.

$ cp data.txt /tmp/myname/dataNew.txt

This will copy data.txt from your current directory to /tmp/myname directory.

$ xxd -r dataNew.txt > data.txt

This will convert your hexdump file back to binary file and will store in a new file naming data.txt .

$ file data.txt

This will tell you that the file is gzip compressed.

$ zcat data.txt > data1.txt

It will uncompress it and store it in new file naming data1.txt . Now same procedure is followed according to the type of file compression.

$ file data1.txt
$ bzcat data1.txt > data2.txt
$ file data2.txt
$ zcat data2.txt > data3.txt
$ file data3.txt
$ tar -xf data3.txt
$ file data5.bin
$ tar -xf data5.bin
$ file data6.bin
$ bzcat data6.bin > data5.txt
$ file data5.txt
$ tar -xf data5.txt
$ file data8.bin
$ zcat data8.bin > data6.txt
$ file data6.txt

You will see the file contains ASCII text.

$ cat data6.txt

You will see you password.
