BANDIT: 05

You need to know about [file]. This tells the type of a file.

Commands:

$ ls

$ cd inhere

You will find so many files here. Some of they may contain some non-readable data.

$ file ./*

Here [*] means all the files. You will find that only -file07 contains ASCII text that is human-readable.

$ cat ./-file07

You will see the password of the next level.
